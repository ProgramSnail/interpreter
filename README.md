# Interpreter

## Language description

#### Variables (def)

name: \#type;

\#int - integer type

#### Conditions

? ( condition ) {

\	do then

} : {

\	do else

}

#### Loops

@ ( condition ) {



}

#### Assignments

name = expression;

#### Function Calls

function_name(arg0, arg1, arg2);

#### Operators

\- a

a + b

a - b

a * b

a / b

a % b

(...)

#### Logical Operators

!a

a | b

a & b

a == b

a != b

(...)

#### Comparasions

a == b

a != b

a < b

a > b

(...)

#### Strings

"string"

\n, \r, \t, \0 - special

\\\\ - \\

#### Constants

$name - constant


$true - true

$false - false

#### Functions

print(integer_value) - print int

write(string) - print string

scan() - read int
